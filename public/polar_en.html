"LED Bar" project

Some information about this project

Project presentation

In order to validate my 2nd year of BTS, I had to choose and carry out with my team a project for the POLAR association. This association is specialized in the realization of plays of light for different types of scenes. The project we have chosen, entitled "LED Bar", consists in realizing a device allowing to control the lighting of a scenography more easily than what the association already has.

What already exists

In order to realize the different scenographies, the association uses a grandMa. This console allows you to configure LED bars placed at different places on the stage. Since the association rents its equipment, it has chosen to create their own LED bar using LED strip light which will then be put into transparent tubes so that the light can be seen. This greatly lowers the cost of LED light bars, but the control of the bars is more difficult as the LED strips lights and grandMa don't use the same protocol to communicate.

Our mission

The purpose of this project is to create a device for grandMa and LED strip lights to communicate with each other in order to achieve any desired configuration.

A potential solution

Conceptual scheme of the project


One solution that our referent professor provided us is to configure a Raspberry to act as an interpreter so that the grandMa and the LED strips can understand each other. To do this, we will configure the Raspberry so that it can interpret DMX frames (frame format used to create lighting effects) that are sent by the grandMa over the network using the Art-Net protocol, which makes it easy to send a DMX frame over the network. Once the Raspberry has received the Art-Net frame, it will be able to extract DMX frames and convert it to the format that the LED strip lights uses.
